Redes de Computadoras

Las redes de computadoras son sistemas interconectados que permiten que múltiples dispositivos,
como computadoras, servidores, impresoras y otros dispositivos, se comuniquen y compartan recursos,
como datos, aplicaciones y servicios. Estas redes facilitan la transferencia de información y recursos
entre los dispositivos, lo que a su vez permite la colaboración y la comunicación eficiente entre usuarios
y sistemas en diversos lugares.

Tipos de Redes

Red de Área Local (LAN - Local Area Network):

Alcance Geográfico: Las LAN son redes de área pequeña y limitada que normalmente se encuentran
en un lugar geográfico reducido, como una oficina, una escuela, un hogar o un edificio.
Propósito: Su finalidad principal es conectar dispositivos cercanos, como computadoras,
impresoras y servidores, para compartir recursos y facilitar la comunicación interna.
Tecnología: Las LAN pueden utilizar conexiones cableadas (como Ethernet) o inalámbricas (Wi-Fi)
para interconectar dispositivos.
Velocidad: Suelen ofrecer velocidades de transferencia de datos muy altas dentro de su alcance limitado.

Red de Área Metropolitana (MAN - Metropolitan Area Network):

Alcance Geográfico: Las MAN cubren un área geográfica más amplia que las LAN, generalmente una ciudad o una región metropolitana.
Propósito: Conectan múltiples LAN en diferentes ubicaciones dentro de un área metropolitana,
lo que permite la interconexión de sucursales de una empresa o instituciones académicas, por ejemplo.
Tecnología: Las MAN suelen utilizar tecnologías como fibra óptica y enlaces de alta velocidad
para conectar las redes locales en diferentes lugares.

Red de Área Extensa (WAN - Wide Area Network):

Alcance Geográfico: Las WAN tienen un alcance mucho más amplio que las LAN y MAN, ya que pueden abarcar ciudades,
países e incluso continentes.
Propósito: Conectan diferentes redes LAN y MAN en ubicaciones geográficamente dispersas.
Permiten la comunicación y el intercambio de datos a larga distancia.
Tecnología: Utilizan diversos medios de transmisión, como líneas dedicadas, fibra óptica, satélites y conexiones de Internet,
para conectar diferentes ubicaciones geográficas.
Velocidad: Las velocidades de transferencia de datos pueden variar según la tecnología utilizada y la
distancia entre las ubicaciones.

Protocolos de Red

Los protocolos de red son un conjunto de reglas y estándares que permiten que los dispositivos en una red se
comuniquen y transfieran datos de manera eficiente y confiable. Estos protocolos establecen cómo se deben enviar,
recibir, verificar y administrar los datos dentro de una red.

Algunos de ellos son:
TCP/IP (Protocolo de Control de Transmisión/Protocolo de Internet)

HTTP (Protocolo de Transferencia de Hipertexto)

HTTPS (Protocolo de Transferencia de Hipertexto Seguro)

FTP (Protocolo de Transferencia de Archivos)

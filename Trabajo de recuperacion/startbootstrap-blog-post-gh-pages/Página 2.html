Instalación de Software

La instalación de programas computacionales (software) es el proceso por el cual nuevos programas
son transferidos a un computador y, eventualmente, configurados, para ser usados con el fin para el cual fueron desarrollados.
Un programa recorre diferentes fases de desarrollo durante su vida útil

Tipos de Software

Software de sistema
Software de Aplicación
Software de Desarrollo
Software de Base de Datos
Software de Seguridad
